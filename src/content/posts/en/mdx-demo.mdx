# MDX Demo

This is a post written in **MDX**, which allows using JSX components in Markdown!

## Inline Component Example

You can embed custom components in your posts:

<div style="padding: 1rem; background: var(--code-bg); border-radius: 6px; margin: 1rem 0;">
  <strong>This is a custom HTML block!</strong>
  <p>You can use any HTML or React components.</p>
</div>

## Code Highlighting

Using Astro's Code component:

<Code code={`function hello() {
  console.log('Hello from MDX!');
}`} lang="javascript" />

## Summary

MDX allows us to:
- ✅ Use Markdown syntax
- ✅ Embed JSX components
- ✅ Import and use external components
- ✅ Create richer interactive content
